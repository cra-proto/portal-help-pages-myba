Submit documents Upload Facility

  1. What is Submit documents?
  2. Who should use Submit documents?
  3. What are the allowable file types?
  4. What is the maximum allowable file size?

1. What is Submit documents?

An Internet based file upload facility provided by CRA that allows small / medium sized businesses to upload their books and financial records.

2. Who should use Submit documents?

Small / medium sized businesses and their representatives that have been contacted by CRA and given instructions on how to upload their financial information using Submit documents.

3. What are the allowable file types?

The following file types will be rejected:

.exe, .ade, .adp, .app, .asx, .bas, .bat, .chm, .class, .cmd, .com, .cpl, .crt, .dll, .fxp, .hlp, .hta, .inf, .ins, .isp, .js, .jse, .lnk, .mda, .mde, .mdt, .mdw, .mdz, .msc, .msi, .msp, .mst, .ops, .pcd, .pif, .prf, .prg, .reg, .scf, .scr, .sct, .shb, .shs, .url, .vb, .vbe, .vbs, .wsc, .wsf, .wsh

4. What is the maximum allowable file size?

Submit documents supports a maximum file size of 150MB. However, the effective maximum file size that can be uploaded will depend on the internet connection speed, due to upload time limitation
