The View return status screen allows you to view the status of returns you have filed with CRA. 

The definition for each of the column headings is as follows:

Reporting year: This is the year for which was reported in the TFSA return and the not the year the return was filed.

Return status:

  • Processed: This means that we have processed the return, regardless if individual records were rejected.
  • In progress: This means we have not fully processed the return.  An error may have been detected in the return and is awaiting correction before we can process it.  We may have already contacted the filer or will be doing so.
  • Rejected: This means that we have rejected the return back to the filer.

Note: If the return has not been received by CRA, it is not displayed.

Submission number: This is the number which was provided to you on the confirmation of receipt page when you filed your return via the Internet File Transfer service.

Summary Sequence number: This is the sequence number in which the return was provided in the submission.

Report type code:

  • Original indicates that the return type code "O" was provided on the return
  • Amended indicates that the return type code "A" was provided on the return

Number of records submitted in the return: This is the total number of individual records that were indicated on the return, as filed.

Date Received: this is the date the file was submitted to the CRA.

Date Processed: this is the date our system processed the return.
