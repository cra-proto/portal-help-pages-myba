Summary/slip information and amendments on My Business Account

View return details (B-VRD-01)

To view the return status, select a reporting year and/or return type to view specific information and click Filter. Otherwise, all reporting years and return types will be displayed.

To file amendments from a specific return, select the link under Return status. To file amendments from all returns, select Search slips.

For large numbers of returns (i.e. over 400), results will not be displayed unless a filter is set.

If a filter for reporting year and return type is not set, you will not be able to search for slips or download returns from this page.

Summary Information (B-VRD-##-02)

This page shows detailed information of the selected summary, including return information, financial information, and general information. Select Search slips to retrieve specific slip information. Select Download to request an electronic version of the return.

Slip information listing (B-VRD-03)

On this Slip information page, you can select individual slips to view the slip details. Select Slip management to amend selected slips in Web Forms.

Slip information details (B-VRD-##-04)

This page contains detailed information on the selected slip, including return information, contributor or recipient information, and financial information. On some slip types the page will also contain slip specific information.

Slip information listing – search (B-VRD-05)

On this page you can search for individual information slips from the return. You can select individual slips to view the slip details. You can also select individual slips to be modified in Web Forms. Selecting the slips to be modified will populate the current slip information in Web Forms.

Slip Management (B-VRD-06)

This Slip Management page displays a listing of the slips that you have selected to be modified in Web Forms. You can also view the details of the slip(s) you have selected.

Download (B-VRD-07)

On this page, you can request a downloadable version of the slips. The report will be in .csv format. Your request will take up to 48 hours to be processed.

The report can be retrieved from the Download reports page under the applicable heading on the My Business Account welcome page. The report will be available online for a period of 10 days.

Note

The maximum number of download requests permitted per day is 50.

Download – Confirmation (B-VRD-08)

This page confirms that we have received your submission.

Your report will be available under the Download reports link of the applicable account type.

Note

Your request will take up to 48 hours to be processed.
