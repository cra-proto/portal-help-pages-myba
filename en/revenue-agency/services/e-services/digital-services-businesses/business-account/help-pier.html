PIERs cases

General information

This page displays a list of PIER cases on file for the payroll deduction account.

Reporting year

Select a reporting year to view only one specific year at a time. Otherwise, all reporting years will be displayed.

PIERs on file

This table displays a list of PIER cases we have on file for the selected reporting year, or all years if no reporting year is selected.

  1. Selecting a link under “Case number” will link to the PIER summary page of the selected case.
  2. Selecting a link under “Download now” will download the selected case file.
  3. Selecting a link under “Reply” will link to the Reply employee listing page of the selected case.

Selecting the “Back” button will return to the previous page in the browser history.

PIER summary

Case information

This table displays the tax year and the case number of the selected PIER case.

Calculated deficiencies for tax year YYYY

These are the amounts we used to calculate the CPP and EI deficiencies for this case.

Selecting the “Back” button will return to the previous page in the browser history.

Selecting the “View employee listing” will display the Employee listing page.

Selecting the “Reply to PIER” page will display the Reply to PIER web form for the selected case.

Employee listing

Slip ### of ###

This section displays the employee information and deficiency information for the given slip.

Employee information

This table displays the identification information for the employee.

Deficiency information

This table displays details of the earnings, deductions reported, deductions required, and deficiencies from the given slip.

Selecting the “View PIER summary” button will display the PIER summary page.
