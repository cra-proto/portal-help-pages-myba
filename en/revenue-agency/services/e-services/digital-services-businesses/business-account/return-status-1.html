1. General information

This page allows you to see general information about your information returns for the current calendar year and the last calendar year.

2. Taxation year

The taxation year is displayed and represents the taxation year for the return processed.

3. Return type

When the status of a return is "processed" it means we have processed at least one return for the return type displayed. Valid values are T4, T4A, T4RSP, T4RIF, T4A-NR, and T1204.

4. Status

Describes the status of the return:

  • Processed
    Means that we have processed a return to our system.
  • Not processed
    Means that we have not processed a return to our system for the taxation year.
