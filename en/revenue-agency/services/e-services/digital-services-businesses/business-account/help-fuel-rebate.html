Before you file

Access the PDF fillable/saveable version of the fuel charge rebate form from the Forms and Publications section at canada.ca/cra-forms. Fill out the fuel charge rebate form and save it to your computer.

How to file

Screen id: B-CT-REF-01

File a rebate

Period covered

Start date (YYYY–MM–DD) (required): Enter the first day of the period covered by this rebate.

End date (YYYY–MM–DD) (required): Enter the last day of the period covered by this rebate.

If you receive this error message "91776 – You cannot electronically file a return/refund for this period. Please refer to the Help section for more information," you must enter more recent period dates or send a paper rebate to:

Canada Revenue Agency
Prince Edward Island Tax Centre
275 Pope Road
Summerside PE  C1N 6E7

Previous: Returns you to the My Business Account homepage.

Next: Takes you to the next step.

Screen id: B-CT-SD-01

Submit fuel charge documents – Upload file

Make sure the file format you want to send is acceptable. The file must be in one of these formats to be uploaded: .pdf, .docx, .xls, .xlsx, .rtf, .txt, .jpg, .jpeg, .tiff, .tif, or .xps. The total size of the file you send must be 150MB or less

Description (max. 254 characters) (required): Describe the file in the space provided.

File (required): Use the "Browse" button in the File field to select the file you want to upload from your computer. If you need to add another file you can do it on the next page.

Upload file: Choose this button after attaching the file to upload your rebate form.

Previous: Returns you to the previous screen.

Screen id: B-CT-SD-02

Submit fuel charge documents – Review

This page displays the file you uploaded but have not yet submitted.

Add another file: If you want to upload another file, click this button. 

Delete file: If you want to delete an uploaded file, check the box next to the file, then choose "Delete file."

Certification statement: You must tick the box certifying that the information you gave is accurate and complete before you can click "Submit." Otherwise, you will not be able to send the rebate.

Submit: Automatically sends the rebate to the Canada Revenue Agency. You will then see a confirmation page.

Previous: Returns you to the previous screen.

Screen id: B-CT-SD-01A

Submit fuel charge documents – Upload file

Note: You will see this screen again only if you selected the "Add another file" button on Page B-CT-SD-02.

Make sure the file format you want to send is acceptable. The file must be in one of these formats to be uploaded: .pdf, .docx, .xls, .xlsx, .rtf, .txt, .jpg, .jpeg, .tiff, .tif, or .xps. The total size of the file you send must be 150MB or less.

Description (max. 254 characters) (required): Describe the file in the space provided.

File (required): Use the "Browse" button in the File field to select the file you want to upload from your computer.

Upload file: Choose this button after attaching the file to upload your rebate.

Previous: Returns you to the previous screen.

Screen id: B-CT-SD-03

Submit fuel charge documents – Confirmation

The confirmation page gives you a confirmation number for your rebate, the filing date, and details of the file(s) you submitted.

Print: This button allows you to print the confirmation page.

After you file

Confirmation number

You will receive a confirmation number after you send your rebate for processing. This confirmation number means the Canada Revenue Agency received your rebate.

You may want to print a copy of the confirmation page for your records.

Do not mail us a paper copy of the rebates you filed online.

After we receive your rebate

We may contact you to verify your supporting documents. It is simply part of our process for all paper and online rebates. This does not mean you are being audited.

Email notifications

Most correspondence is only available online in My Business Account by default, except when a business has changed its delivery method to receive paper mail. If you are registered for My Business Account, you can add an email address to receive notifications when you have mail in your online account. The CRA will also email you to let you know if important changes are made to your account.

To register, select “Profile” then “Manage notification preferences” and follow the steps.
