Deficiencies and explanations

Slip ### of ###

This section displays the employee information and deficiency information for the given slip.

Employee information

This table displays the identification information for the employee.

Deficiency information

This table displays details of the earnings, deductions reported, deductions required, and deficiencies from the given slip. New amounts can be entered for employment income (box 14), pensionable earnings (box 26), insurable earnings (box 24), CPP deductions reported (box 16), and EI deductions reported (box 18).

Explanation code

This drop-down contains a list of common reporting errors. An explanation code must be selected if new amounts are entered in the deficiency information table.

Explanation

If “(12) Other” is selected from the explanation code drop-down box, a text box will be displayed to enter details of the explanation for the deficiency. The text box is limited to 154 characters and can be left blank.

Selecting the “Clear changes” button will reset all changes (new amounts, explanation text) to blank.

Selecting the “View reply summary” button will display the Reply summary page.

Selecting the “Submit reply” button displays the Reply summary page to allow the reply to be reviewed and confirmed.

Selecting the “PIER overview” page displays the PIER overview page.

Reply summary

Employee information

This table displays the identification information for the employee.

Deficiency information

This table displays details of the earnings, deductions reported, deductions required, and deficiencies from the given slip. New amounts can be entered for employment income (box 14), pensionable earnings (box 26), insurable earnings (box 24), CPP deductions reported (box 16), and EI deductions reported (box 18).

Explanation code

This drop-down contains a list of common reporting errors. An explanation code must be selected if new amounts are entered in the deficiency information table.

Explanation

If “(12) Other” is selected from the explanation code drop-down box, a text box will be displayed to enter details of the explanation for the deficiency. The text box is limited to 154 characters and can be left blank.

Person to contact, area code, telephone number, extension

It is mandatory to enter the name of a business contact and telephone number.

Selecting the “Confirm reply” button submits the PIER reply to the CRA and displays the Confirmation of reply page.

Selecting the “View more slips” button returns to the Employee listing page.

Selecting the “Back” button will erase any changed data, return to the previous page in the browser history and will not submit the reply to CRA. A warning message will be displayed.

Confirmation of receipt

Employee information

This table displays the identification information for the employee.

Deficiency information

This table displays details of the earnings, deductions reported, deductions required, and deficiencies from the given slip. It lists both the amounts reported and the amounts from the PIER reply.

Explanation code

This is the code that was selected to explain the deficiency.

Explanation

If “(12) Other” was selected as the explanation code, the text entered as the explanation will be displayed.

Person to contact

The name and telephone number of the person entered as the contact will be displayed.

Selecting the “PIER overview” button displays the PIER overview page.

Selecting the “Save” button will save the entered explanations and the confirmation number in pdf format.

Selecting the “End session” button will close the PIER session without saving changes and display the My Business Account welcome page.
