Q1. What is Submit documents?

A1. Submit documents is a secure online service the CRA provides that allows businesses to upload their requested documents.

Q2. Who should use Submit documents?

A2.  Businesses and their representatives who the CRA has contacted and given instructions on how to upload their supporting documents.

Q3. What are the allowable file types?

A3. The CRA will accept the following file types:
.pdf, .doc, .docx, .xls, .xlsx, .rtf, .txt, .jpg, .jpeg, .tiff, .tif, .xps

Q4. What is the maximum total file upload size?

A4. You can submit multiple documents during a session. The total file size of all submitted documents cannot exceed 150MB.

Q5. Will the documents be scanned for viruses?

A5. The CRA will scan all submitted files for viruses. We will reject any files infected with a virus and will contact you with instructions for submitting the file again using a different method.

B-RC-SD-02: Submit documents – Upload file

  • Description – Please enter a description of the document you are attaching (maximum 254 characters).
  • Tax year-end – A drop-down menu will appear when more than one tax year-end is applicable. Please select the tax year-end that corresponds to the document.
  • Browse – Select ‘Browse’ and a window will appear to allow you to select the file to attach. 
  • Upload file – This will allow you to upload a file.
  • Cancel – This will return you to the welcome page.

B-RC-SD-02A: Submit documents – Upload file

  • Description – Please enter a description of the document you are attaching (maximum 254 characters).
  • Tax year-end – A drop-down menu will appear when more than one tax year-end is applicable. Please select the tax year-end that corresponds to the document.
  • Browse – Select ‘Browse’ and a window will appear to allow you to select the file to attach. 
  • Upload file – This will allow you to upload a file.
  • Cancel – This will return you to the welcome page.

B-RC-SD-03: Submit documents - Review

  • Delete file(s) – Allows you to delete the selected file(s). Please note that if you are deleting the file, you will have to attach the file again.
  • Add another file – Allows you to return to the Upload file page.
  • Confirm – Allows you to accept the submission of the files.
  • Cancel – This will return you to the welcome page.

B-RC-SD-04: Submit documents – Confirmation

  • Make another submission – This will return you to the Submit documents page.
